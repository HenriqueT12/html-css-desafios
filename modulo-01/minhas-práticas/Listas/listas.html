Definições de alguns termos e tecnologias da informática

HTTP
A sigla HTTP na verdade é um protocolo que deve ser seguido para que as informações das páginas web sejam transmitidas pela rede
Links
Links são âncoras que tem como função fazer o usuário "viajar" pela web, entre artigos e sites. As âncoras são clicáveis, e o usuário pode clicar se desejar visitar uma outra pagina web, ou visitar um artigo que tenha explicações das quais o usuário procura.
URLs
Uma URL é o endereço de um site hospedado em um servidor web.
