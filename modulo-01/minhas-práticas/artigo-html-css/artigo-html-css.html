HTML e CSS

Definição de HTML

HTML ou Linguagem de HiperTexto e Marcação é uma tecnologia web com ituito de apresentar marcações em textos e outras informações na internet.
Com ela é possível contruir suas páginas web, e contribuir seus conhecimentos e suas informações com o mundo!. Pois ela permite que os navegadores entendam as marcações feitas que chamamos de tags, elas dão ênfase semântica ao conteúdo que digitamos em nossas páginas web.

Lista de coisas possíveis em HTML

  • Demarcar o conteúdo de forma semântica.
  • Demarcar títulos de artigos, paragrafos.
  • Demarcar palavras que tenham ênfase em seus artigos.
  • Demarcar âncoras, ou links, que interligam seu conteúdo a outros artigos ou sites.
  • Demarcar tags que carregam imagens para seu site.

Definição de CSS

As CSS ou Folhas de Estilo em Cascata, é uma tecnologia que coopera com a HTML, ela é utilizada para fazermos nossas formatações de tipos de fontes, cores, espaçamentos de conteúdos e muito mais!. Ou com ela podemos criar os nossos proprios design de conteúdo, que será exibido para os leitores de nossos sites.

Listas de coisas possíveis com CSS

  • Estilizar fontes e tamanhos de textos.
  • Mudar a cor de fundo das páginas web.
  • Definir tamanho de imagens, e blocos de conteúdos dentro do site.
  • Definir a posição de conteúdo dentro das páginas web.
